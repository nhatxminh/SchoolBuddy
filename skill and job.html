Skills and Jobs

1.Overview

Fundamentally, School buddy is a highly friendly and modern school management system which has been working on as a project for 10 weeks. All the functionalities and inspirations have been fabricated all by the first-year students of the RMIT students. This project was observed to be promising by many instructors and contributors. During this time, we would like to request assistance from supporters with medium/high experience in the specific fields. The participants who can understand and adapt with a certain number of tasks and requirements to help this project achieve the max potential in the upcoming years.

Front - end Developer:

Duties and responsibilities:

  • Collaborating with backend developers to create Friendly-Interface and helpful applications/website.
  • Creating website with codes like CSS, JavaScript (dynamite using different libraries).
  • Always take responsible for any complain, change or mistake arise.
  • Turning the website accessible to many platforms (IOS, Android, Window, etc.)
  • Incorporating animations, graphic like videos and illustrations into the website.
  • Deliver fast.

Requirements:

  • Fluency with both verbal and written English.
  • Bachelor’s/Master’s degree in computer science, IoT (or similar experience)
  • At least 3-5 years of experience collaborating with User Interface
  • Medium/effective communication skills (collaborating with Backend developers)
  • Experienced with JavaScript, CSS (and React)
  • Good at adapting and understanding customer’s need
  • Stay professionally overseeing problems despite in stressful situations
  • Competitive attitude is a high plus
  • Able to work through Git

Back - end Developer:

Duties and responsibilities:

  • Provide and maintain reliable API which function smoothly with Frontend
  • Optimize server for the best experience and stability
  • Ensure that the website can operate all the functions
  • Always analyze and enhance the system if it is possible
  • Capable of investigating and debugging if there is any error occurs

Requirements:

  • Fluency with both verbal and written English
  • 3+ years of experience working with backend development
  • Experience in AWS, NodeJS, SQL, Python
  • Effective in teamwork and communication
  • High adaptability if there are any changes in project
  • Friendly and ready to teach, rise any participants working in the project

Technical Lead:

Duties and responsibilities:

  • Setting schedules and tasks for the team
  • Record the progress and deliver clearly to the member about the status
  • Identify risks and prepare a plan to minimize the consequences
  • Keep updating the technical trends and developments
  • Checking goals weekly and monthly

Requirements:

  • Bachelor’s degree in computer science and engineering, IoT.
  • Management certification and experience is highly recommended
  • Capability of leading and controlling the group in dispute situations
  • Motivation in work and able to inspire other
  • Always stay professional and discipline in work

Cybersecurity:

Duties and responsibilities:

  • Be able to identify suspicious activities and quickly solve security breaches
  • Prevent data thief by using different methods like firewall, data-encryption, etc.
  • Proactive in updating and improving security programs
  • Carry out regular ethical hacking
  • Ready to instruct other employees to avoid fraud, phishing, etc.

Requirements:

  • Bachelor’s degree in IT or equivalent bachelor
  • Deep understanding about security about firewall, proxy server, SIEM, etc.
  • Well-built investigate skills
  • Sharp and innovative mindset
  • Be able to perform well under pressure
